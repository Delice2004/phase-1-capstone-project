BookVerse – Capstone Project

BookVerse is a responsive web application that allows users to search books using the Open Library API, view book details, and save their favorite books using localStorage.

The app includes three main pages: Home (search books), Favorites (saved books), and About (project information).

It features dynamic book search, favorite management (add/remove/clear), and a clean responsive UI built with HTML, Tailwind CSS, and JavaScript (ES6 modules). Git and GitHub were used for version control with feature branches and pull requests.

 Features
Search books using Open Library API
Display book title, author, cover, and publish year
Add/remove books from favorites
Save data using localStorage
View favorite books with stats
Responsive design for all devices

 Tech Stack

HTML, Tailwind CSS, JavaScript (ES6 Modules), Open Library API, Git & GitHub
